# WorldGaming Frontend

Frontend construido con React + Vite + TypeScript + TailwindCSS.

## Requisitos
- Node 18+
- npm 9+

## Desarrollo
`ash
npm install
npm run dev
`

## Build
`ash
npm run build
`

## Estructura
- src/app: módulos de la app
- shared: componentes y servicios compartidos
